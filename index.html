API Key Required

To use this standalone translator, you need to configure your Gemini API Key.

How to configure your API key:

  1. In the Settings Area: Click the settings gear icon in the top header and paste your key, then click "Save Key". This saves the key in your browser.
  2. In the Source Code: Alternatively, open this index.html file in VS Code and replace the placeholder variable value inside the script: const GEMINI_API_KEY = "YOUR_API_KEY";

Get your key for free at Google AI Studio.
